# travisgerard.com

Simple static personal site for writing and the source dictations behind selected pieces.

The source writing lives outside this repo in:

```text
../life/Writing
```

Only files listed in `content.json` are built into the public site. That keeps drafts and variants from being published accidentally.

## Build

```bash
npm run build
```

## Local Preview

```bash
npm run serve
```

Then open:

```text
http://localhost:8777
```
